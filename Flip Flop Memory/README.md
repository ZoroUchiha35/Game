# Gameplay Overview
This is a standard memory pair-matching game with a clean, modern interface.

- **Core Rules:** Flip cards to reveal animal emojis. Find two matching emojis to keep them face-up. The game tracks your **total moves** and how many **pairs** you've completed.
- **Interaction:** Click any card to start. If two flipped cards don't match, they will turn back over after a short delay. A message area provides clear feedback `("Match found!", "Not a match")`.
