# 🎮 Game Collection

Welcome to my personal game collection! This repository contains browser-based games that you can play instantly. No downloads, no installations – just pure gaming fun!

## 🎯 Live Demo

Play all games here: **https://zorouchiha35.github.io/Game/**

#### How to Play
1. Click the link then choose the game you want
2. Start playing immediately
3. No downloads needed

## 📁 Games Available

|     Game                 |         Description                                    |
|--------------------------|--------------------------------------------------------|
| 🐦 **Flappy Bird**      | Tap to fly, avoid pipes. The classic arcade challenge.  |
| 🧠 **Flip Flop Memory** | Test your memory. Match pairs, train your brain.        |
| 🏓 **Ping Pong**        | Hit the ball past your opponent to score. Classic two-player arcade fun.  |

## ✨ Features

- **100% Free** – No ads, no paywalls, no tracking
- **Works Everywhere** – Desktop, tablet, mobile
- **Instant Play** – No loading screens, no waiting
- **Lightweight** – Each game is a single HTML file
- **Open Source** – Code is here for you to explore
